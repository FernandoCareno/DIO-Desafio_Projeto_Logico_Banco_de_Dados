# DIO-Desafio_Projeto_Logico_Banco_de_Dados
# Projeto Banco de Dados E-commerce

## Descrição

Projeto lógico de banco de dados para um cenário de e-commerce.

## Regras de Negócio

- Cliente pode ser PF ou PJ (não pode ser ambos).
- Pedido pode possuir vários produtos.
- Pedido pode possuir múltiplas formas de pagamento.
- Entrega possui status e código de rastreio.
- Produto pode estar em vários estoques.
- Produto pode ser fornecido por vários fornecedores.

## Estrutura

- Script de criação do banco
- Script de inserção de dados
- Queries avançadas com:
  - SELECT
  - WHERE
  - ORDER BY
  - HAVING
  - JOIN
  - Atributos derivados
